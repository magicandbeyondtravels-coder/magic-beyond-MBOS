# Core

The `core` domain holds business identity: mission, vision, values, brand philosophy, positioning and voice. It is the most upstream domain in the repository (see ARCHITECTURE.md section 5) — every other domain traces back to the objects here, and nothing in `core` references downstream domains. Start with `mission.md`, then `vision.md` and `values.md`, before reading the brand-specific objects.

All objects below are placeholders (`status: draft`) awaiting full content per the Phase 1 roadmap in ARCHITECTURE.md section 18.
