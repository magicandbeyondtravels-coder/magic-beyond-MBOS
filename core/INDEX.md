# Core

The `core` domain holds business identity: mission, vision, values, brand philosophy, positioning and voice. It is the most upstream domain in the repository (see ARCHITECTURE.md section 5) — every other domain traces back to the objects here, and nothing in `core` references downstream domains. Start with `positioning.md` for the executive summary, then follow its `related` links out to the canonical objects it points to.

`positioning.md` is an executive summary: it states only what is unique to positioning and references the canonical objects below rather than duplicating them, per [ADR-0002](../decisions/2026-07-04-canonical-knowledge-objects.md). `vision.md`, `messaging.md` and `values.md` remain placeholders (`status: draft`) awaiting approved content.
